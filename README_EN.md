## ğŸ— The Base Code Pipeline (src/)
We used a modular structure for the final project:

* **`data_loader.py`**: Loads data.
* **`preprocessing.py`**: Cleans data and handles outliers.
* **`features.py`**: Feature engineering.
* **`visualization.py`**: Plots key insights.
* **`models.py`**: **Your Playground!** Implement your ML models here.
* **`main.py`**: Runs the entire pipeline.
